📝 To-Do List App

A modern and user-friendly To-Do List application developed using Python and Tkinter GUI.
This application helps users organize daily tasks efficiently with a clean interface and productivity sections.


---

✨ Features

✅ Add New Tasks
✅ Delete Selected Tasks
✅ Clear All Tasks
✅ Notes Section
✅ Focus Section
✅ Reminder Section
✅ Modern GUI Design
✅ Easy-to-Use Interface


🛠️ Technologies Used

Python

Tkinter GUI

📁 Project Structure

todo-list-app/
│
├── todo.py
